Laboratoriumuitwisseling ()
Laboratoriumgegevens op uniforme wijze digitaal uitwisselen, zodat zorgverleners, patiënten, laboratoriumspecialisten en onderzoekers gegevens eenduidig interpreteren. Dit voorkomt fouten en bespaart tijd. Handmatig overtypen is bijvoorbeeld niet meer nodig. https://nictiz.nl/standaarden/informatiestandaarden/labuitwisseling/
