Koppeltaal services ()
Koppeltaal services zijn stukjes waardevolle functionaliteit voor een eindgebruiker. - Een eindgebruiker kan een persoon zijn; zoals een zorgverlener, een patiënt, of een koppeltaal-beheerder. - Een eindgebruieker kan ook een andere applicatie zijn. Gezien de hechte samenwerking leveren de op Koppeltaal aangesloten applicaties diverse services aan elkaar.
