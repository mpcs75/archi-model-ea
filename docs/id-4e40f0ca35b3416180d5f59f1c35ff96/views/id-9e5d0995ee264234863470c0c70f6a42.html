01. Toepassing e-health GGZ ()
Deze hybride zorgreis is gebaseerd op de generieke e-health module van AKWA GGZ. De hybride zorgreis GGZ draait om de mogelijkheden en kansen die er zijn bij het ontvangen van digitale zorg door een goede inzet van e-health.
